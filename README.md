# Loan_online
## Итоговый проект Антона Воронкина.
***
## **Реализованный функционал:**
- Логин и регистрация пользователя
- Список займов
- Просмотр детальной информации о займе
- Оформление займа
- Выход из аккаунта 
- Кэширование данных

## **Стек технологий:**
- Kotlin coroutines
- ViewModel, LiveData
- Koin
- Retrofit, Okhttp
- Room
- Navigation component

## **Архитектура**
- Clean Architecture
- MVVM
